# PPBO08 – Polimorfisme
L0324007 – Atikah Dhiya

Project ini berisi empat latihan mengenai konsep dasar polimorfisme pada Java.  
Setiap latihan diletakkan dalam satu package terpisah.

## Struktur Project
Latihan1  
Latihan2  
Latihan3  
Latihan4  

## Latihan 1
Membuat kelas Animal, Dog, dan Bird.  
Method speak() dioverride pada Dog dan Bird.  
Objek disimpan dalam List<Animal> dan dipanggil speak() secara polymorphic.  
Dog memiliki method tambahan fetch().

## Latihan 2
Membuat abstract class Shape dengan method area().  
Mengimplementasikan Circle dan Rectangle.  
Menghitung total area dari seluruh objek Shape dalam satu list.

## Latihan 3
Membuat interface Payable.  
Mengimplementasikan Employee (gaji tetap) dan Contractor (dibayar per jam).  
Menghitung total pembayaran dari semua objek Payable.

## Latihan 4
Menguji perbedaan overriding dan overloading.  
Base memiliki method f(Number n).  
Sub memiliki method f(Integer i), sehingga ini overloading.  
Ketika memanggil b.f(10) pada Base b = new Sub(), method yang dipanggil milik Base.

## Cara Menjalankan
1. Buka project di NetBeans.  
2. Jalankan class Main pada masing-masing Latihan sesuai kebutuhan.
